# The Indus Group Website

A modern, responsive reference website developed to showcase a corporate business website with multiple service pages and a clean user interface.

## Features

- Responsive design
- Multi-page website
- Modern navigation
- Business service pages
- Interactive JavaScript functionality
- Mobile-friendly layout
- Clean and organized project structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Purpose

This project was created as a reference website for learning, practice, and demonstration of front-end web development concepts.
